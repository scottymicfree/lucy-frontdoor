## Where the real system lives
Lucy’s canon, blueprint vault, and implementation live in a **private repository**.
This repo intentionally contains **no source code** or internal specs.

## What Lucy is
Lucy is a **sovereign cognitive operating system concept** with:
- a distributed cognitive mesh model
- an oversight/safety supervisor layer
- a single external voice discipline
- a UI concept for visibility and governance

## What this repo is NOT
- Not an API server
- Not an SDK
- Not the implementation
- Not a place to submit PRs for core behavior
